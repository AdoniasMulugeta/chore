# vue-js-projects
### Chore is a simple to-do app build with vue.js, bootstrap-vue and firebase

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
